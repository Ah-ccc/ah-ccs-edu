# 更改SYSCONFIG版本

因为下载的空历程不是实时更新的

所以有时候下载的空历程是用的老版本的SYSCONFIG

如何切换：

1. 选中项目

2. 点开 project

![alt text](image-1.png)

3. 点开properties

![alt text](image-2.png)

4. 左侧栏通用下面选择dependencies

![alt text](image-2.png)

    点击sysconfig

5. 选择版本

![alt text](image-3.png)

6. 切换完成之后重新打开sysconfig
